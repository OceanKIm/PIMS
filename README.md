# project_issue_manage_system
## [개인포트폴리오] 프로젝트 이슈 관리 시스템

 ※프로젝트 개요
 PIMS  (project issue manage system) 
  : 오픈 프로젝트 관리 시스템

> 해당 프로젝트는 실제 업무에 주어진  과제를 수행함에 있어서
> 사내의 협업 능률을 높이고자 과제 업무 중  발생한 이슈를 로그화 하고
> 게시판을 활용하게 하는 시스템 


- ERD CLOUD (현재 설계중)
